
I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  Cur_PC     (9 bits)
 - input  Imm        (32 bits)
 - input  JalrSel
 - input  Branch
 - input  AluResult  (32 bits)
 - output PC_Imm     (32 bits)
 - output PC_Four    (32 bits)
 - output BrPC       (32 bits)
 - output PcSel

The module should be purely combinational and implement the basic branch
unit functions:

 - PC_Imm: The PC-relative target computed as Cur_PC + Imm.
   Treat Cur_PC as an unsigned PC value zero-extended to 32 bits before
   addition. Imm is a 32-bit signed immediate; perform a 32-bit addition.

 - PC_Four: The sequential PC computed as Cur_PC + 32'd4 (with Cur_PC
   zero-extended to 32 bits prior to the addition).

 - BrPC: The selected branch/jump target address. First compute
   Branch_Sel = Branch && AluResult[0]. If JalrSel=1, choose AluResult
   (e.g., JALR target). Otherwise, if Branch_Sel=1, choose PC_Imm
   (e.g., branch or JAL target). If neither JalrSel nor Branch_Sel is
   asserted, set BrPC to 32'd0.

 - PcSel: Set to 1 when a control-flow change is taken; otherwise 0.
   For this problem, a branch is taken only when Branch=1 and AluResult[0]=1.
   JALR is taken whenever JalrSel=1. Therefore, PcSel = JalrSel ||
   (Branch && AluResult[0]).
